METHODS

Methods Summary

Data processing. Paired-end reads were trimmed with Trim Galore (v0.6) using --2colour 20. Bisulfite-converted reads were aligned to hg38 with bwameth (v0.2.7). Duplicates were marked with sambamba (v0.8). Methylation ratios were extracted with MethylDackel using minimum depth 10× and --maxVariantFrac 0.25. Nucleosome occupancy was computed with DANPOS3; WPS was computed with a 120 bp window.

Differential analysis. Mann-Whitney U tests were applied per CpG site. Multiple testing was corrected by Bonferroni (α = 0.05 / 3,771,981 ≈ 1.33×10⁻⁸).

Power analysis. Per-CpG statistical power was estimated using NormalIndPower (statsmodels). Read-depth-dependent missing rates were modelled by a negative binomial distribution fitted to empirical TWIST panel data. Per-CpG σ was derived from 497 reference samples.

MESA modeling. Each cfDNA feature modality was independently processed through MESA_modality (missing-value filtering → variance filtering → univariate selector → Boruta feature selection → modality predictor). Modality-level out-of-fold probabilities were stacked by a logistic regression meta-estimator. Cross-validation used leave-one-out (LOOCV). Performance was evaluated by ROC-AUC.
